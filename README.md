# changescale
Tiny script for fast raising or lowering "Typing to piano" scales for FL Studio 20
```
Usage: changescale [input_file] [output_file] [amount]
```
# Example
```
> changescale ".\input_file.scr" ".\output_file.scr" 12
```
Input file:
```
C4
D4
E4
F4
FREE
FREE
G4
A4
B4
```
Output file:
```
C5
D5
E5
F5
FREE
FREE
G5
A5
B5
```
